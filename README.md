# Uebung-021  --  Leap Year Checker

## Lernziele:
- Boolsche Variablen
- Modulo

## Aufgabenstellung:
Schreiben Sie Ein C# Programm, welches überprüft ob das eingegebene Jahr ein Schaltjahr ist.  
Eingaben < 0 sind als fehlerhafte Eingabe zu behandeln.
